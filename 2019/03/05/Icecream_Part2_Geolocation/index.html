具体可以通过浏览器的 Geolocation API 来实现。在app.js 脚本中，使用 navigator.geolocation 对象来访问此API。 navigator.geolocation 对象有这样几个方法：getCurrentPosition()，watchPosition()，clearWatch()， 可以帮助我们实现开始的需求。
